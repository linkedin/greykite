Getting Started

It won't take too long to get your first forecast with Greykite. Proceed with the following steps:

  1. Set up your environment and get Greykite installed: installation guide.
  2. Follow the quick start example to play with Greykite on the Peyton Manning wiki pageview data simple forecast example. Now you are able to generate your first forecast.
  3. Read the forecast process guide to understand how forecasting happens in the Greykite framework.
  4. Follow the tuning tutorial try what you learned: tuning tutorial.
  5. Browse more quick start examples and tutorials to learn more functionalities and get more insights.
  6. Now that you are an "expert", enjoy forecasting your own datasets with Greykite!
